# floor-planner

## Draw Tools

These are the core tools users always expect when drawing a floor plan.

1. Wall Tools

   - Straight Wall
   - Continuous Wall (multi-segment)
   - Curved Wall (optional)
   - Room (auto-wall loop)
   - Delete Wall / Split Wall / Extend Wall

2. Opening Tools

   - Door
   - Single door
   - Double door
   - Sliding door
   - Window
   - Standard window
   - Sliding window
   - Bay/bow window (optional)

3. Structural Tools

   - Columns / Pillars
   - Beams
   - Floor Levels (optional)
   - Stairs (optional but very useful)

4. Measurement Tools

   - Dimension Line
   - Linear (horizontal/vertical)
   - Aligned (angled)
   - Room Area Measurement
   - Distance Measure Tool (temporary measure)

5. Annotation Tools

   - Text / Labels
   - Arrows
   - Notes
   - Symbols (e.g., north sign, electrical symbols)

6. Furniture Tools (if you support objects)

   - Furniture Library Picker
   - Sofa, bed, table, etc.
   - Kitchen Objects
   - Bathroom Objects
   - Custom Object Insert
   - Image Import (as texture/reference)

7. Shapes & Drafting Tools

   - These help users sketch before wall placement.
   - Rectangle
   - Circle
   - Line
   - Polygon / Custom shape
   - Snap controls (toggle snapping)

8. Editing / Manipulation Tools
   - (Not drawing, but often placed under Draw)
   - Select / Move
   - Rotate
   - Scale
   - Duplicate
   - Delete
   - Group / Ungroup (optional)
   - Lock / Unlock elements
